    HuggingFaceInferenceProvider¶

    wordbase.providers.huggingface_inference ¶

    HuggingFace Inference API provider for cloud-hosted models.

    This module implements the APIProvider interface for the HuggingFace Inference API, which allows running any model on the Hub without local hardware. It uses the OpenAI-compatible chat completions endpoint exposed by InferenceClient.

    The provider handles
    • Authentication via explicit API key or HF_TOKEN env var
    • Message format conversion (OpenAI-style role/content dicts)
    • Per-token log probability extraction when the model supports it
    • Thinking content extraction from ... blocks
    • Concurrent n > 1 generation via ThreadPoolExecutor
    • Token counting via a lazy-loaded tokenizer, with character-based fallback when transformers is not installed

    Typical usage example:

    from wordbase.providers.huggingface_inference import HuggingFaceInferenceProvider
from wordbase.types import Message

provider = HuggingFaceInferenceProvider(
    model="meta-llama/Llama-3.1-8B-Instruct"
)
messages = [Message(role="user", content="Hello!")]
completions = provider.generate(messages, temperature=0.7)
print(completions[0]["text"])

    HuggingFaceInferenceProvider ¶

    HuggingFaceInferenceProvider(model: str, api_key: str | None = None, endpoint_url: str | None = None)

    Bases: APIProvider

    Provider for the HuggingFace Inference API.

    Uses InferenceClient's chat_completion() endpoint, which follows the OpenAI chat format. Supports both serverless inference (public Hub models) and dedicated endpoints.

    Inherits generate() and generate_batch() from APIProvider. The HF Inference API does not support a native n parameter, so n > 1 requests are handled by n concurrent API calls.

    Per-token log-probabilities are available for models that expose them. When requested but unavailable, a UserWarning is emitted and the 'tokens' list is left empty.

    def __init__(
    self,
    model: str,
    api_key: str | None = None,
    endpoint_url: str | None = None,
) -> None:
    """Initialize the HuggingFace Inference API provider.

    Args:
        model: HuggingFace model ID (e.g. "meta-llama/Llama-3.1-8B-Instruct")
            or any model accessible through the Inference API.
        api_key: HuggingFace API token. Falls back to the HF_TOKEN env
            var when omitted.
        endpoint_url: URL of a dedicated Inference Endpoint. When
            provided, the model ID is used only for metadata; requests
            are routed to this URL instead of the Hub serverless API.

    Raises:
        ImportError: If huggingface-hub is not installed.
        AuthenticationError: If no API key is found in the argument or
            the HF_TOKEN environment variable.
    """
    super().__init__()
    self.model = model
    self.endpoint_url = endpoint_url

    resolved_key = self._resolve_api_key(api_key, "HF_TOKEN")

    if endpoint_url:
        self._client: InferenceClient = InferenceClient(
            base_url=endpoint_url, token=resolved_key
        )
    else:
        self._client = InferenceClient(token=resolved_key)

    # Lazy-loaded tokenizer for count_tokens().
    self._tokenizer: Any = None
    self._tokenizer_loaded: bool = False

    def count_tokens(self, text: str) -> int:
    """Count tokens using a lazy-loaded tokenizer.

    Attempts to load the model's tokenizer via ``transformers`` on the
    first call. Falls back to a character-based approximation
    (1 token ≈ 4 characters) when ``transformers`` is unavailable or the
    tokenizer cannot be loaded, emitting a UserWarning in that case.

    Args:
        text: Text to tokenize and count.

    Returns:
        Token count, exact when a tokenizer is available or approximate
        otherwise (minimum 1).
    """
    if not self._tokenizer_loaded:
        try:
            from transformers import (  # pylint: disable=import-outside-toplevel
                AutoTokenizer,
            )

            self._tokenizer = AutoTokenizer.from_pretrained(self.model)
        except Exception:  # pylint: disable=broad-except
            self._tokenizer = None
        self._tokenizer_loaded = True

    if self._tokenizer is None:
        warnings.warn(
            f"Cannot load tokenizer for '{self.model}'. "
            f"Using character-based approximation "
            f"({_CHARS_PER_TOKEN} chars ≈ 1 token). "
            "Install wordbase[huggingface] for accurate token counting.",
            UserWarning,
            stacklevel=2,
        )
        return max(1, len(text) // _CHARS_PER_TOKEN)

    return len(self._tokenizer.encode(text))
